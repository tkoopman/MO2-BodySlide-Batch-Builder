BodySlide Batch Builder (BSBB)

About Help Nexus Link

About

This is a plug-in for MO2 that will run BodySlide automatically for multiple configured builds.

Prerequisites

  • Mod Organizer 2
  • BodySlide and Outfit Studio
  • Executable configured in MO2 (Ctrl+E) called "BodySlide x64".
    This is what automatically gets added when you have BodeSlide installed in MO2 normally.

Installation

Extract the downloaded file into your MO2\plugins folder. So you should have a folder x:\...\MO2\plugins\BSBB\

NOTE: If you using v1.x of BSBB you should uninstall it first. It was a single file x:\...\MO2\plugins\BodySlideBatchBuilder.py
Configuration will not migrate from v1.x to v2 of BSBB as it was a complete re-design to try and make it more user friendly.
Any future upgrades should keep configuration.
